# HireSense AI

AI-Powered Resume Shortlisting and Candidate Ranking System

## Overview

HireSense AI is an intelligent HR automation platform that:
- extracts resume information
- parses job descriptions
- evaluates candidates
- ranks applicants
- generates shortlist reports

Built using Python, Streamlit, NLP concepts, and rule-based AI scoring.

---

## Features

- Resume Parsing
- JD Parsing
- Candidate Scoring
- Candidate Ranking
- PDF Report Generation
- Human Override Panel
- Analytics Dashboard
- Interactive Charts

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- PyMuPDF
- ReportLab

---

## System Architecture

1. Upload JD
2. Upload Resumes
3. Extract Text
4. Parse Skills
5. Score Candidates
6. Rank Candidates
7. Generate Reports
